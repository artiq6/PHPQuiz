
# PHPQuiz

Simple quiz game with accoutns, admin panel etc.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Installing

1
```
copy repository
```
2
```
uznip, run xampp
```
3
```
import database as 'quiz'
```
4
```
run localhost/quiz
```

## Built With

* JS
* HTML
* CSS
* BOOTSTRAP (v.4.0)
* JQuery
* MYSQL
